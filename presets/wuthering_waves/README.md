# 鸣潮 (Wuthering Waves) — 自动化预设

## 游戏信息

- **游戏名称**: 鸣潮 / Wuthering Waves
- **开发商**: Kuro Games
- **窗口标题**: `鸣潮` 或 `Wuthering Waves`
- **窗口类名**: `UnrealWindow`
- **分辨率要求**: 1920×1080（全屏）

## 前置条件

1. **游戏已安装并正常运行**
2. **系统设置**:
   - 分辨率：1920×1080
   - 显示模式：全屏
3. **游戏内设置**:
   - 语言：简体中文
   - 键盘映射：默认键位（W/A/S/D 移动，F 互动，E 共鸣技能，Q 共鸣解放，M 地图，J 任务日志，T 钩索）
   - 已解锁活跃度系统和每日任务功能
4. **工具要求**:
   - Python 3.10+
   - EasyOCR 或 Windows OCR 已安装
   - 以管理员权限运行（pyautogui 权限要求）

## 启动步骤

1. 启动鸣潮，进入游戏主界面
2. 打开「全能脚本」工具
3. 选择「鸣潮」→「每日任务」
4. 确认角色在主城安全区域
5. 点击「开始执行」

## 预设说明

### daily_mission — 每日任务自动流程

**流程概览**:
```
打开活跃度界面 → 领取活跃度奖励 → 一键领取 → 打开任务日志 
→ 选择每日任务 → 追踪任务 → 传送 → 移动至目标 → 战斗 
→ 完成确认 → 领取奖励 → 循环
```

**执行时长**: 每轮约 2-4 分钟

**节点类型覆盖**:
- ✅ `wait` — 等待加载
- ✅ `press_key` — 键盘按键（移动/技能/闪避/钩索）
- ✅ `key_hold` — 组合键长按
- ✅ `mouse_click` — 鼠标点击
- ✅ `click_text` — OCR 识别并点击文字
- ✅ `condition` — 条件分支（检查剩余任务数）
- ✅ `loop` — 循环节点（多轮执行）

**错误处理**:
- 超时重试：关键节点支持最多 3 次重试
- 错误跳过：非关键节点出错后继续执行
- 传送失败自动重试

## 注意事项

⚠️ **重要提醒**

1. **不要手动操作**: 脚本运行期间请勿移动鼠标或按键
2. **网络稳定**: 传送加载时间受网络影响，网络不佳时手动增加等待时间
3. **战斗风险**: 建议使用等级 70+ 角色执行每日任务
4. **钩索使用**: 部分地图可能无法使用钩索，脚本在失败后会自动跳过
5. **版本更新**: 游戏版本更新后 UI 可能变动，需重新截取模板图片
6. **安全提示**: 自动化工具违反游戏用户协议，使用风险自负

# 鸣潮 (Wuthering Waves) 模板特征描述

> **说明**: 此目录存放鸣潮预设所需的 UI 元素模板图片。
> 由于游戏版本迭代，以下为特征描述，实际使用时可按需截取对应模板。

## 所需模板清单

| 模板名称 | 描述 | 用途 | 截图区域 |
|---------|------|------|---------|
| 活跃度图标 | 屏幕右上角的活跃度图标 | 打开活跃度奖励界面 | 右上角 HUD 区域 |
| 领取按钮 | 金色"领取"按钮 | 领取活跃度奖励 | 活跃度界面 |
| 一键领取 | "一键领取"按钮（蓝色） | 批量领取奖励 | 活跃度界面底部 |
| 任务日志 | 任务界面的"每日"标签 | 查看每日任务 | 任务日志界面顶部 |
| 追踪按钮 | "追踪"按钮 | 追踪当前任务 | 任务详情界面 |
| 传送锚点 | 地图上的传送锚点图标（蓝色菱形） | 传送到目标区域 | 地图界面 |
| 互动提示 | "F 互动"提示文字 | 与 NPC/物品互动 | 屏幕中央下方 |
| 钩索图标 | 钩索准星提示 | 使用钩索移动 | 屏幕中央 |
| 完成提示 | "完成"文字提示 | 检测任务完成 | 屏幕中央 |
| 拾取提示 | "拾取"或"F 拾取"文字 | 拾取掉落物 | 屏幕中央下方 |

## 模板制作指南

1. 使用截图工具截取游戏界面中对应元素
2. 裁剪至仅包含目标元素（留少量边距）
3. 保存为 PNG 格式，放入此目录
4. 文件命名使用英文驼峰命名，如 `activityIcon.png`
5. 建议模板尺寸：80×80 ~ 200×60 像素
6. 匹配阈值: 默认 0.8（可在设置中调整）

## OCR 识别说明

- 游戏内中文使用自定义字体，OCR 识别语言设置为 `chi_sim`
- 建议使用 EasyOCR 或 Windows OCR 引擎
- 关键识别文字: "活跃度"、"领取"、"一键领取"、"每日"、"追踪"、"传送"、"互动"、"完成"、"拾取"

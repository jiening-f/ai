# 绝区零 (Zenless Zone Zero) 模板特征描述

> **说明**: 此目录存放绝区零预设所需的 UI 元素模板图片。
> 由于游戏版本迭代，以下为特征描述，实际使用时可按需截取对应模板。

## 所需模板清单

| 模板名称 | 描述 | 用途 | 截图区域 |
|---------|------|------|---------|
| 每日图标 | 主界面的"每日"入口图标 | 打开每日任务界面 | 主界面菜单栏 |
| 领取按钮 | 黄色"领取"按钮 | 领取每日奖励 | 每日界面 |
| 一键领取 | "一键领取"按钮 | 批量领取 | 每日界面底部 |
| VR训练入口 | "VR训练"入口标识 | 进入VR训练设施 | 录像店内 |
| 材料关卡 | "材料"关卡图标 | 选择材料刷取 | VR训练关卡列表 |
| 开始挑战 | "开始挑战"按钮 | 开始副本 | 关卡详情界面 |
| 通关提示 | "通关"文字提示 | 检测副本完成 | 结算界面顶部 |
| 确认按钮 | "确认"按钮 | 确认退出结算 | 结算界面 |
| 再次挑战 | "再次挑战"按钮 | 重复刷取 | 结算界面底部 |
| 退出按钮 | "退出"按钮 | 退出VR训练 | VR训练界面 |

## 模板制作指南

1. 使用截图工具截取游戏界面中对应元素
2. 裁剪至仅包含目标元素（留少量边距）
3. 保存为 PNG 格式，放入此目录
4. 文件命名使用英文驼峰命名，如 `dailyIcon.png`
5. 建议模板尺寸：80×80 ~ 200×60 像素
6. 匹配阈值: 默认 0.8（可在设置中调整）

## OCR 识别说明

- 游戏内中文使用自定义UI字体，OCR 识别语言设置为 `chi_sim`
- 建议使用 EasyOCR 或 Windows OCR 引擎
- 关键识别文字: "每日"、"领取"、"一键领取"、"VR训练"、"材料"、"开始挑战"、"通关"、"确认"、"再次挑战"、"退出"

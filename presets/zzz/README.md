# 绝区零 (Zenless Zone Zero) — 自动化预设

## 游戏信息

- **游戏名称**: 绝区零 / Zenless Zone Zero
- **开发商**: miHoYo / HoYoverse
- **窗口标题**: `绝区零` 或 `Zenless Zone Zero`
- **窗口类名**: `UnityWndClass`
- **分辨率要求**: 1920×1080（全屏）

## 前置条件

1. **游戏已安装并正常运行**
2. **系统设置**:
   - 分辨率：1920×1080
   - 显示模式：全屏
3. **游戏内设置**:
   - 语言：简体中文
   - 键盘映射：默认键位（W/A/S/D 移动，F 互动，E 特殊技，Q 终结技，Shift 闪避，Space 切换代理人）
   - 已解锁"VR训练"功能和"每日任务"
4. **工具要求**:
   - Python 3.10+
   - EasyOCR 或 Windows OCR 已安装
   - 以管理员权限运行（pyautogui 权限要求）

## 启动步骤

1. 启动绝区零，进入录像店主界面
2. 打开「全能脚本」工具
3. 选择「绝区零」→「日常刷取」
4. 确认在录像店内
5. 点击「开始执行」

## 预设说明

### daily_farm — 日常刷取流程

**流程概览**:
```
打开每日界面 → 领取每日奖励 → 一键领取 → 进入VR训练 
→ 选择材料关卡 → 开始挑战 → 战斗（攻击/技能/闪避/连携/终结技） 
→ 通关结算 → 领取奖励 → 再次挑战/退出 → 循环
```

**执行时长**: 每轮约 2-3 分钟

**节点类型覆盖**:
- ✅ `wait` — 等待加载/结算
- ✅ `press_key` — 键盘按键（技能/闪避/切换/终结技）
- ✅ `mouse_click` — 鼠标连点（基础攻击）
- ✅ `click_text` — OCR 识别并点击文字
- ✅ `condition` — 条件分支（循环/退出判断）
- ✅ `loop` — 循环节点（多轮刷取）
- ✅ `sequence` — 战斗序列（攻击→技能→闪避→连携→终结技循环）

**错误处理**:
- 超时重试：关键节点支持最多 2 次重试
- 错误跳过：非关键节点出错后继续执行（如一键领取无内容时跳过）
- 通关检测超时：等待 60 秒后自动重试

## 注意事项

⚠️ **重要提醒**

1. **不要手动操作**: 脚本运行期间请勿移动鼠标或按键
2. **体力限制**: VR 训练消耗体力，请确保有足够体力
3. **战斗风险**: 建议使用等级 60+ 代理人队伍
4. **连携提示**: 脚本会自动按下 Space 触发连携，请勿在设置中关闭连携提示
5. **版本更新**: 游戏版本更新后 UI 可能变动，需重新截取模板图片
6. **安全提示**: 自动化工具违反游戏用户协议，使用风险自负

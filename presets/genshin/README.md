# 原神 (Genshin Impact) — 自动化预设

## 游戏信息

- **游戏名称**: 原神 / Genshin Impact
- **开发商**: miHoYo / HoYoverse
- **窗口标题**: `原神` 或 `Genshin Impact`
- **窗口类名**: `UnityWndClass`
- **分辨率要求**: 1920×1080（全屏）

## 前置条件

1. **游戏已安装并正常运行**
2. **系统设置**:
   - 分辨率：1920×1080
   - 显示模式：全屏（或无边窗口）
   - 渲染精度：1.0
3. **游戏内设置**:
   - 语言：简体中文
   - 键盘映射：默认键位（W/A/S/D 移动，F 互动，E 元素战技，Q 元素爆发，M 地图，F1 冒险之证）
   - 已完成新手引导，解锁"冒险之证"功能
4. **工具要求**:
   - Python 3.10+
   - EasyOCR 或 Windows OCR 已安装
   - 以管理员权限运行（pyautogui 权限要求）

## 启动步骤

1. 启动原神，进入游戏主界面（可操作角色状态）
2. 打开「全能脚本」工具
3. 选择「原神」→「每日委托」
4. 确认角色在蒙德/璃月/稻妻/须弥等主城安全区域
5. 点击「开始执行」

## 预设说明

### daily_commission — 每日委托自动流程

**流程概览**:
```
打开冒险之证 → 领取每日奖励 → 追踪委托 → 打开地图 
→ 传送至目标区域 → 移动到战斗点 → 战斗 → 领取奖励 → 循环
```

**执行时长**: 每轮约 2-5 分钟（取决于委托类型）

**节点类型覆盖**:
- ✅ `wait` — 等待加载
- ✅ `press_key` — 键盘按键
- ✅ `key_hold` — 组合键长按
- ✅ `mouse_click` — 鼠标点击
- ✅ `click_text` — OCR 识别并点击文字
- ✅ `condition` — 条件分支
- ✅ `loop` — 循环节点

**错误处理**:
- 超时重试：关键节点（如打开冒险之证）支持最多 3 次重试
- 错误跳过：非关键节点（如拾取掉落物）出错后继续执行
- 传送失败自动重试

## 注意事项

⚠️ **重要提醒**

1. **不要手动操作**: 脚本运行期间请勿移动鼠标或按键，避免干扰执行
2. **网络稳定**: 传送加载时间受网络影响，网络不佳时请手动增加等待时间
3. **战斗风险**: 角色等级过低可能导致战斗失败，建议使用等级 80+ 角色
4. **树脂消耗**: 此预设不消耗树脂，仅完成每日委托
5. **版本更新**: 游戏版本更新后 UI 可能变动，需重新截取模板图片
6. **安全提示**: 自动化工具违反游戏用户协议，使用风险自负。建议仅在单人模式下使用

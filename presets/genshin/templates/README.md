# 原神 (Genshin Impact) 模板特征描述

> **说明**: 此目录存放原神预设所需的 UI 元素模板图片。
> 由于游戏版本迭代，以下为特征描述，实际使用时可按需截取对应模板。

## 所需模板清单

| 模板名称 | 描述 | 用途 | 截图区域 |
|---------|------|------|---------|
| 冒险之证 | 屏幕右上角的书本图标（金色） | 打开每日委托界面 | 右上角 HUD 区域 |
| 委托标签 | 冒险之证界面中的"委托"标签页 | 切换到委托列表 | 冒险之证界面上方 |
| 领取奖励 | 黄色"领取奖励"按钮 | 领取每日委托奖励 | 冒险之证界面右下 |
| 追踪 | 蓝色"追踪"按钮 | 追踪当前委托目标 | 委托详情右侧 |
| 传送锚点 | 地图上的蓝色传送锚点图标 | 传送至任务区域 | 地图界面 |
| 传送按钮 | 传送确认弹窗中的"传送"按钮 | 确认传送 | 传送确认弹窗 |
| 挑战开始 | 红色"挑战"文字提示 | 检测战斗开始 | 屏幕中央偏上 |
| 完成 | "完成"文字提示 | 检测委托完成 | 屏幕中央 |
| 再次挑战 | "再次挑战"按钮 | 检测完成后的继续 | 结算界面中央 |
| 拾取 | "拾取"或"F 拾取"提示 | 拾取掉落物 | 屏幕中央下方 |

## 模板制作指南

1. 使用截图工具截取游戏界面中对应元素
2. 裁剪至仅包含目标元素（留少量边距）
3. 保存为 PNG 格式，放入此目录
4. 文件命名使用英文驼峰命名，如 `adventureHandbook.png`
5. 建议模板尺寸：80×80 ~ 200×60 像素
6. 匹配阈值: 默认 0.8（可在设置中调整）

## OCR 识别说明

- 游戏内中文使用方正准圆等字体，OCR 识别语言设置为 `chi_sim`
- 建议使用 EasyOCR 或 Windows OCR 引擎
- 关键识别文字: "冒险之证"、"委托"、"领取奖励"、"追踪"、"传送"、"挑战"、"完成"、"再次挑战"、"拾取"

# PresetEditor 预设编辑

## 用途
管理某个游戏下的所有预设配置，编辑预设的步骤和参数。

## 布局结构

```
┌──────────────────────────────────────────────────────────────┐
│  预设管理 > 原神                             [+ 新建预设]    │  ← 面包屑 + 操作按钮
├──────────────────────────────────────────────────────────────┤
│                                                              │
│  搜索框 [______________]        [全部状态 ▼]                 │  ← 筛选栏
│                                                              │
│  ┌──────────────────────────────────────────────────────────┐│
│  │ 预设名称    │ 描述       │ 步骤数 │ 状态      │ 操作    ││
│  ├──────────────────────────────────────────────────────────┤│
│  │ 自动采集    │ 采集白铁矿  │ 12    │ ● 启用    │ [✏][▶] ││  ← 预设列表
│  │ 日常任务    │ 每日委托    │ 8     │ ● 启用    │ [✏][▶] ││
│  │ 周本刷怪    │ 刷周本材料  │ 15    │ ○ 禁用    │ [✏][▶] ││
│  │ ├─ 深渊挑战  │ 深境螺旋   │ 20    │ ○ 禁用    │ [✏][▶] ││  ← 缩进表示分组
│  │ 自动烹饪    │ 批量烹饪    │ 5     │ ● 启用    │ [✏][▶] ││
│  └──────────────────────────────────────────────────────────┘│
│                                                              │
│  共 5 条                                    < 1/1 >          │  ← 分页（可选）
│                                                              │
└──────────────────────────────────────────────────────────────┘
```

## 区块说明

| 区块 | 组件 | 说明 |
|------|------|------|
| 面包屑 | Breadcrumb | "预设管理 > 原神"，点击"预设管理"回到游戏选择 |
| 筛选栏 | Input + Select | 搜索框 + 状态筛选（全部/启用/禁用） |
| 预设列表 | Table | 显示预设名称、描述、步骤数、状态开关、操作按钮 |

## 交互细则

- 状态列使用 Switch/Toggle 开关，切换即时生效
- 操作列：[✏] 编辑跳转到 FlowEditor，[▶] 立即执行
- 行内编辑：点击预设名称可直接重命名（inline input）
- 拖拽排序：支持拖拽改变预设顺序
- 点击描述可展开编辑

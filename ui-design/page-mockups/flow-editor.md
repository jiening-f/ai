# FlowEditor 流程编辑器

## 用途
可视化拖拽编排脚本流程，支持 19 种节点类型。

## 区块说明

| 区块 | 宽度 | 说明 |
|------|------|------|
| 节点面板 | 280px | 左侧固定，节点按分类分组（流程/键盘/鼠标/视觉/数据），拖拽到画布 |
| 画布区域 | 弹性 | ReactFlow 画布，网格背景，缩放/平移控制 |
| 属性面板 | 320px | 右侧固定，选中节点后显示可编辑属性 |
| 底部日志栏 | 100% | 可折叠/展开，显示实时执行日志 |

## 节点面板详情

- 每个分类可折叠展开
- 节点项显示图标 + 名称
- 拖拽时显示半透明预览
- 搜索框可筛选节点名称

## 画布交互

- 鼠标滚轮缩放，中键拖拽平移
- 点击节点选中（高亮边框）
- 从节点端口拖出连线到另一节点
- 右键节点显示上下文菜单（复制/删除/禁用）
- 框选多个节点可批量移动/删除
- 运行中当前节点高亮闪烁

## 属性面板详情

- 根据选中节点类型动态显示不同表单
- 按键节点：按键选择器 + 延迟设置
- 条件节点：条件表达式编辑器
- 等待节点：时间/条件选择
- 底部显示"应用更改"按钮

# GameManager 游戏管理

## 用途
管理已配置的游戏列表，添加/删除/编辑游戏配置。

## 布局结构

```
┌──────────────────────────────────────────────────────────────┐
│  游戏管理                                [+ 添加游戏]        │  ← 顶部栏 + 操作按钮
├──────────────────────────────────────────────────────────────┤
│                                                              │
│  ┌──────────┐  ┌──────────┐  ┌──────────┐                   │
│  │ 🎮 原神   │  │ 🎮 鸣潮   │  │ 🎮 绝区零  │  ← 游戏卡片网格  │
│  │  window:  │  │  window:  │  │  window:  │                   │
│  │  原神     │  │  鸣潮     │  │  绝区零    │                   │
│  │  3 预设   │  │  2 预设   │  │  1 预设   │                   │
│  │          │  │          │  │          │                   │
│  │ [编辑]   │  │ [编辑]   │  │ [编辑]   │                   │
│  │ [删除]   │  │ [删除]   │  │ [删除]   │                   │
│  └──────────┘  └──────────┘  └──────────┘                   │
│                                                              │
```

## 区块说明

| 区块 | 组件 | 说明 |
|------|------|------|
| 页面标题 | 文字 | "游戏管理" 标题 + 右侧 "[+ 添加游戏]" Primary 按钮 |
| 游戏卡片网格 | Card（Game 变体） | 响应式网格（min-width: 280px, gap: 20px），每张卡片显示游戏名、窗口信息、预设数量 |
| 添加卡片 | Card | 特殊卡片：虚线边框，居中 "+" 图标，点击打开添加对话框 |

## 游戏卡片内容

- 顶部 120px 渐变色块（根据游戏名称生成渐变色）
- 游戏名称（h3）
- 窗口标题 / 窗口类（small 文字）
- 预设数量 Badge（primary 色）
- 底部操作按钮（编辑 Secondary / 删除 Ghost-Danger）

## 添加/编辑游戏

- 点击添加/编辑打开 Modal
- 表单字段：游戏名称、窗口标题、窗口类
- 保存后卡片即时出现

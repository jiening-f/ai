## 区块说明

| 区块 | 组件 | 说明 |
|------|------|------|
| 筛选栏 | Input + Select × 2 | 搜索 + 分类 + 状态筛选 |
| 插件卡片网格 | Card（Plugin 变体） | 响应式网格（min-width: 260px, gap: 20px）|
| 安装插件 | Modal | 支持上传 .zip 或填入插件仓库 URL |

## 插件卡片内容

- 左上角 40px 插件图标（或默认 🔌 emoji）
- 插件名称（h3）
- 版本号（small，`--text-tertiary`）
- 描述文字（body，最多 2 行截断）
- 状态 Badge（已启用/已禁用/未安装）
- 操作按钮（根据状态动态显示）

## 交互细则

- 启用/禁用开关即时生效
- 卸载需二次确认 Modal（"确定卸载插件 XXX？"）
- 安装支持拖拽 .zip 文件到页面
- 插件卡片悬停显示阴影提升

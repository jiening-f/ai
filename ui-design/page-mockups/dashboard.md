# Dashboard 仪表盘

## 用途
用户登录后的首页，提供整体运行状态概览和快捷操作入口。

## 布局结构

```
┌──────────────────────────────────────────────────────────────┐
│  仪表盘                                             用户头像 │  ← 顶部栏
├──────────────────────────────────────────────────────────────┤
│                                                              │
│  ┌──────┐  ┌──────┐  ┌──────┐  ┌──────┐                    │
│  │ ● 运行 │  │ ● 暂停 │  │ ● 停止 │  │ ● 总计 │  ← 状态统计卡片  │
│  │  3    │  │  1    │  │  5    │  │  9    │                    │
│  └──────┘  └──────┘  └──────┘  └──────┘                    │
│                                                              │
│  ┌─────────────────────────┐  ┌─────────────────────────┐    │
│  │  快捷操作                │  │  最近执行记录             │    │
│  │                         │  │                         │    │
│  │  [▶ 运行预设]           │  │  原神 - 自动采集   3m前  │    │
│  │  [□ 新建预设]           │  │  鸣潮 - 日常任务   15m前 │    │
│  │  [⚡ 截图识别]          │  │  绝区零 - 刷本    1h前  │    │
│  │  [✏ 编辑流程]           │  │  ...                     │    │
│  └─────────────────────────┘  └─────────────────────────┘    │
│                                                              │
│  ┌────────────────────────────────────────────────────┐      │
│  │  执行趋势（最近 7 天）                               │      │
│  │  ▁▃▂▅▄▇▆    ← 柱状图区域                            │      │
│  │  一 二 三 四 五 六 日                                │      │
│  └────────────────────────────────────────────────────┘      │
│                                                              │
└──────────────────────────────────────────────────────────────┘
```

## 区块说明

| 区块 | 组件 | 说明 |
|------|------|------|
| 状态统计卡 | Card（Stat 变体） | 4 个水平排列的统计卡片，显示运行/暂停/停止/总计数量 |
| 快捷操作 | Card + Button | 4 个主要操作按钮，垂直排列，左侧带图标 |
| 最近执行 | Card + Table（简化） | 最近 5 条执行记录，显示预设名、游戏、执行时间、状态 Badge |
| 执行趋势 | Card | 7 天执行次数柱状图（简化为 CSS 柱状），每天一根 |

## 间距

- 统计卡片之间间距 `--space-md`（16px）
- 快捷操作与最近执行之间间距 `--space-lg`（24px）
- 区块内边距统一 `--space-lg`（24px）

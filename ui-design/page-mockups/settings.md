# Settings 系统设置

## 用途
配置应用的全局设置和偏好。

## 布局结构

```
┌──────────────────────────────────────────────────────────────┐
│  系统设置                                                     │  ← 顶部栏
├──────────────────────────────────────────────────────────────┤
│                                                              │
│  [通用] [视觉识别] [快捷键] [关于]      ← Tab 切换          │
│                                                              │
│  ┌─ 通用设置 ────────────────────────────────────────────┐   │
│  │                                                       │   │
│  │  ┌───────────────────────────────────────────────────┐│   │
│  │  │ 语言                          [简体中文 ▼]       ││   │
│  │  │ 界面语言                                          ││   │
│  │  └───────────────────────────────────────────────────┘│   │
│  │                                                       │   │
│  │  ┌───────────────────────────────────────────────────┐│   │
│  │  │ 主题                      [暗色 ▼]               ││   │
│  │  │ 选择界面主题（暗色/亮色/跟随系统）                    ││   │
│  │  └───────────────────────────────────────────────────┘│   │
│  │                                                       │   │

│  │  │ 自动保存间隔                    [ 5 ] 分钟         ││   │

│  │  └───────────────────────────────────────────────────┘│   │
│  │                                                       │   │
```

## 区块说明

| 区块 | 组件 | 说明 |
|------|------|------|
| Tab 导航 | Tabs（线型） | 将设置按类别分组，当前显示"通用"标签页 |
| 设置项卡片 | Card | 每个设置项占一行（大设置项独占卡片） |
| 表单控件 | Input/Select/Toggle | 根据设置类型使用不同的控件 |
| 操作按钮 | Button | "保存更改" Primary / "重置" Ghost |

## 各 Tab 内容

### 通用
- 语言选择（Select）
- 主题切换（Select）
- 自动保存间隔（Number Input）
- 复选框选项（Toggle）

### 视觉识别
- OCR 引擎选择（Select: PaddleOCR / EasyOCR / 自动）
- 匹配阈值滑块（Range: 0.6 - 0.95）
- 默认截图格式（Select: PNG / JPG）
- 截图保存路径（Input + 文件夹选择器）

### 快捷键
- 启动/停止脚本（组合键输入）
- 截图快捷键
- 显示/隐藏窗口
- 每个快捷键可自定义（Input 带 key capture）

### 关于
- 版本号
- 检查更新按钮
- 开源许可证信息
- 开发者信息

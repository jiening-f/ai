## 用途
查看所有脚本的执行记录，分析成功/失败情况和耗时。

## 区块说明

| 区块 | 组件 | 说明 |
|------|------|------|
| 筛选栏 | Select × 3 + Input | 游戏/预设/状态下拉 + 搜索框，时间范围日期选择 |
| 执行列表 | Table | 状态列使用 Badge（完成/失败/停止/运行中） |
| 执行详情 | Card（展开面板） | 点击表格行展开，显示执行详情表格 |

## 交互细则

- 筛选条件变更时列表即时刷新
- 点击行头展开/收起详情
- 失败行红色高亮，光标移到失败行显示错误 Tooltip
- 可选择多行批量删除
- 导出按钮：导出 CSV

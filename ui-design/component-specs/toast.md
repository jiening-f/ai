# Toast 通知

## 用途
非侵入式地反馈操作结果或系统消息。

## 视觉样式

| 属性 | 值 |
|------|-----|
| 背景 | `--bg-elevated` |
| 边框 | `--border-default` |
| 圆角 | `--radius-md`（8px）|
| 阴影 | `--shadow-xl` |
| 内边距 | 12px 16px |
| 最小宽度 | 320px |
| 最大宽度 | 480px |

### 类型变体

| 类型 | 左侧图标色 | 用途 |
|------|-----------|------|
| **success** | `--color-success` ✅ | 操作成功 |
| **error** | `--color-error` ❌ | 操作失败 |
| **warning** | `--color-warning` ⚠️ | 需要关注 |
| **info** | `--color-info` ℹ️ | 一般信息 |

## 结构

```
┌─────────────────────────────────┐
│ [图标] 通知标题              [✕] │
│       通知描述文字（可选）       │
│                                │
│        ────────────────         │  ← 进度条（自动关闭时）
└─────────────────────────────────┘
```

### 各部分

| 部分 | 样式 |
|------|------|
| **图标** | 左侧 20px，与类型色一致 |
| **标题** | `--text-primary`，14px，`--weight-medium` |
| **描述** | `--text-secondary`，12px |
| **关闭按钮** | 右上角 Ghost 风格，16x16 |

## 位置

- 默认: 右上角固定定位
- 间距: 顶部 16px，toast 之间 8px

## 交互细则

- 自动关闭时长：success 3s / info 4s / warning 5s / error 不过期
- 底部进度条显示自动关闭倒计时
- 悬停时暂停自动关闭倒计时
- 进入动画：从右侧滑入（250ms ease）
- 离开动画：向右滑出 + 淡出（200ms ease）
- 最多同时显示 5 个，超出后新 toast 排队

# 游戏全能脚本 — 桌面应用图标设计规格 v3

## 设计概念

**带花茎和叶子的完整淡黄色花朵**。

扁平风格。白色圆角方形背景配可见灰色外框（#bbb），确保在白色桌面上清晰可辨。
花瓣用较深的黄色（#ffd84d）配棕色描边（#8b7355），大幅提升对比度。
花茎和叶子使用绿色系，与花朵形成自然配色。

## 图标规范

| 属性 | 值 |
|------|-----|
| 尺寸 | 512×512 px（源 SVG 可缩放） |
| 形状 | 圆角方形（r=96px / 512px） |
| 风格 | 扁平，适中的对比度 |
| 背景 | 白色 + 灰色外框（#bbb） |

## 色彩体系

```
图标外框:     #bbbbbb      浅灰（确保白色桌面可见）
花瓣底色:     #ffd84d      亮黄
花瓣描边:     #8b7355      深棕（增强对比）
花蕊外层:     #ffb300      橙黄
花蕊中层:     #ff8f00      深橙
花蕊中心:     #e65100      焦橙
花蕊装饰:     #cc7700      点缀色
茎叶主色:     #7cb342      草绿
茎叶描边:     #558b2f      深绿
叶脉:         #558b2f      半透明叶脉
```

## 图标结构

```
┌─────────────────────┐
│   白色背景+灰色外框    │
│       ┌─────┐       │
│      │花瓣×5 │      │
│       └─────┘       │
│         │           │
│       花茎          │
│      ↙     ↘       │
│    叶子    叶子      │
└─────────────────────┘
```

- 花瓣 5 瓣，72° 均匀分布，椭圆 50×72，距中心 105px
- 花蕊三层（r=40, 26, 14），由外到内渐深
- 茎从花心延伸至底部，宽 7-10px
- 左叶位于茎中上部，右叶位于茎中下部

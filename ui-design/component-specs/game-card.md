# GameCard 游戏卡片

## 用途
在游戏管理页面中以卡片形式展示单个游戏的信息，支持点击进入预设管理。

## 视觉样式

| 属性 | 暗色值 | 亮色值 |
|------|--------|--------|
| 背景 | `--color-surface` (#1a1a1e) | `--color-surface` (#ffffff) |
| 边框 | `--color-border`, 1px solid | 同上 |
| 圆角 | `--radius-lg` (12px) | 同上 |
| 悬停效果 | translateY(-2px), shadow-md | 同上 |
| 过度动画 | 150ms ease-out | 同上 |

### 结构

```
┌──────────────────────────────────┐
│ ┌──────────────────────────────┐ │
│ │       渐变色 Header 区        │ │  ← 100px，游戏名称hash色
│ │      [状态徽章]    [图标]      │ │     含半透明图标 + 状态指示
│ └──────────────────────────────┘ │
│                                  │
│  游戏名称                    ←   │  ← 16px, font-weight: 600
│  🪟 窗口标题                ←   │  ← 13px, secondary 文字
│  ────────────────────────────    │  ← 分割线
│  📝 5 个预设    ⏱️ 2 分钟前      │  ← 13px, 灰色文字
│                                  │
│  ────────────────────────────    │  ← thin border
│  [编辑] [运行]          [删除]   │  ← 操作按钮
└──────────────────────────────────┘
```

### 各部分

| 部分 | 说明 |
|------|------|
| **Header** | 100px 渐变色块，颜色根据游戏名称自动生成（固定映射）。内含状态徽标和游戏图标。 |
| **状态徽标** | 右上角 absolute 定位，带毛玻璃背景。取值：`运行中`（绿色脉冲圆点）、`暂停`（黄色）、`离线`（灰色） |
| **游戏图标** | 44×44 圆角方块，半透明毛玻璃背景，emoji 图标 |
| **Body** | 20px 内边距，含名称、窗口标题、元数据行 |
| **元数据行** | 预设数量 + 最后运行时间，由分割线与上方隔开 |
| **操作栏** | 底部浅灰分割线上方，三个按钮：编辑(Ghost)、运行(Ghost)、删除(Danger-right) |

### 交互状态

| 状态 | 操作 |
|------|------|
| 悬停 | 卡片上移 2px，阴影提升至 md，边框变亮 |
| 点击 | 跳转到预设管理页面 (`/presets/{gameId}`) |
| 按钮悬停 | 编辑/运行 → 背景浮现；删除 → 红色背景浮现 |

### 尺寸

| 属性 | 值 |
|------|-----|
| 卡片最小宽度 | 280px |
| 卡片完整高度 | 约 260px |
| 响应式网格 | `repeat(auto-fill, minmax(300px, 1fr))` |
| 网格间距 | 20px |

# AddGameDialog 添加游戏弹窗

## 用途
用于在游戏管理页面中添加新游戏配置的模态对话框。

## 视觉样式

| 属性 | 暗色值 | 亮色值 |
|------|--------|--------|
| 背景 | `--color-surface-raised` (#222226) | `--color-surface-raised` (#fafafa) |
| 边框 | `--color-border`, 1px solid | 同上 |
| 圆角 | `--radius-xl` (16px) | 同上 |
| 阴影 | `--shadow-xl` | 同上 |
| 宽度 | 480px | 同上 |
| 动画 | scale(0.95) + translateY(8px) → scale(1), 300ms ease-out | 同上 |

### 遮罩

| 属性 | 值 |
|------|-----|
| 背景 | rgba(0, 0, 0, 0.6) |
| 毛玻璃 | backdrop-filter: blur(4px) |
| z-index | 400 |

### 结构

```
┌──────────────────────────────────────┐
│  添加游戏                       [✕]   │  ← modal-header
│──────────────────────────────────────│
│                                       │
│  游戏名称  *                          │  ← form-group
│  ┌────────────────────────────────┐   │
│  │ 例如：原神                     │   │  ← input
│  └────────────────────────────────┘   │
│  输入游戏的中文名称                    │  ← form-hint
│                                       │
│  窗口标题                             │  ← form-group
│  ┌────────────────────────────────┐   │
│  │ 窗口标题（可选）                │   │  ← input
│  └────────────────────────────────┘   │
│  留空则自动匹配窗口                    │  ← form-hint
│                                       │
│  窗口类名                             │  ← form-group
│  ┌────────────────────────────────┐   │
│  │ 窗口类名（可选）                │   │  ← input
│  └────────────────────────────────┘   │
│  指定窗口类名以提高匹配精度             │  ← form-hint
│                                       │
│──────────────────────────────────────│
│            [取消]  [保存并添加]        │  ← modal-footer
└──────────────────────────────────────┘
```

### 行为交互

| 操作 | 行为 |
|------|------|
| 点击遮罩 | 关闭弹窗 |
| 按 Esc | 关闭弹窗 |
| 点击「取消」 | 关闭弹窗，不保存 |
| 点击「保存并添加」 | 验证表单 → 调用 API → 成功则关闭 + Toast 提示 |
| 表单验证 | 游戏名称为必填项，为空时阻止提交 |

### 表单字段

| 字段 | 类型 | 必填 | 说明 |
|------|------|------|------|
| 游戏名称 | 文本 | 是 | 游戏的中文名称 |
| 窗口标题 | 文本 | 否 | 游戏窗口标题文字，用于窗口匹配 |
| 窗口类名 | 文本 | 否 | 窗口类名（如 UnityWndClass），提高匹配精度 |

### 状态

| 状态 | 表现 |
|------|------|
| 默认 | 表单为空，placeholder 提示 |
| 聚焦 | 输入框边框变为主色 + 3px 主色 glow |
| 必填验证失败 | 保存时名称若为空，按钮无反应（前端拦截） |
| 保存中 | 按钮 disabled + loading 态 |
| 保存成功 | 关闭弹窗 + Toast "游戏已添加" |
| API 错误 | Toast 展示错误信息 |

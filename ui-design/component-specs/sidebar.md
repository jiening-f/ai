# Sidebar 侧边栏

## 用途
主导航系统，提供应用内所有页面的入口。

## 视觉样式

| 属性 | 值 |
|------|-----|
| 宽度 | 240px（展开）/ 64px（折叠）|
| 背景 | `--bg-sidebar` |
| 右边框 | `--border-default`，1px solid |
| 高度 | 100vh |

## 结构

```
┌──────────────────┐
│  Logo / 应用名   │  ← sidebar-header (64px)
├──────────────────┤
│  ● 仪表盘        │
│  ● 游戏管理      │
│  ● 预设管理      │  ← nav-item
│  ● 流程编辑器    │
│  ● 执行历史      │
│  ● 插件管理      │
│  ● 系统设置      │
├──────────────────┤
│  [折叠按钮]      │  ← sidebar-footer
└──────────────────┘
```

### 导航项

| 状态 | 样式 |
|------|------|
| 默认 | 行高 44px，内边距 12px 20px，文字 `--text-secondary` |
| 悬停 | 背景 `rgba(255,255,255,0.06)`，文字 `--text-primary` |
| 激活 | 左侧 3px `--color-primary` 竖条，背景 `--color-primary-bg`，文字 `--color-primary` |
| 禁用 | 文字 `--text-tertiary`，不可点击 |

### 图标

- 每个导航项左侧 20px 图标
- 折叠态只显示图标，隐藏文字
- 图标色与文字色保持一致

### 分隔线

- 导航分组之间使用 1px `--border-default` 分隔线
- 上下间距各 8px

### 品牌区

- 应用 Logo（24px）+ 应用名 "全能脚本"
- 折叠态只显示 Logo

## 折叠态

- 宽度缩小至 64px
- Tooltip 悬停显示导航项全名
- 切换按钮在底部
- 动画 250ms ease

# 游戏全能脚本 — 桌面应用图标设计规格 v5

## 设计概念

**油画风格侧视花朵，浓密层叠，向右倾斜。**

基于 v4 的油画写实风格进一步优化：
- 花瓣密度大幅提升（16 → 26片），花朵更饱满浓密
- 背景颜色进一步减淡，极浅暖色画布
- 整花向右倾斜 15°，姿态更生动
- 花心居中布局，画面平衡更好

## 图标规范

| 属性 | 值 |
|------|-----|
| 尺寸 | 512×512 px（源 SVG 可缩放） |
| 形状 | 圆角方形（r=96px / 512px） |
| 风格 | 油画写实，极浅暖色调，15° 倾斜构图 |
| 构图 | 花朵侧视朝向右侧，整体倾斜，居中 |

```
中间花瓣:     #e8c840 → #8a6815   金色（主体层）

叶子高光:     #8aac3e              叶脉光泽
图标外框:     #c8b090 → #dcc8aa   浅暖金描边
```

## 图层结构

```
┌──────────────────────────┐
│  极浅暖色画布（更淡）     │
│  ┌────────────────────┐  │
│  │  背景层花瓣（8片）    │  │  ← 暗金色，花头后方
│  │  中间层花瓣（10片）   │  │  ← 金色，花主体
│  │  前景层花瓣（8片）    │  │  ← 亮金色，视觉焦点
│  │     花萼（绿色）      │  │  ← 花头基部
│  └────────────────────┘  │
│         │                │
│       花茎（弯曲）        │
│      ↙     ↘             │
│    叶子    叶子           │
│  （左）    （右）         │
└──────────────────────────┘
```

## 关键改动（v4 → v5）

| 项目 | v4.1 | v5 |
|------|------|----|
| 背景色 | `#f5ead6 → #d4bea0` | `#fffcf5 → #f5dfbd` |
| 花瓣数量 | 16 片 | 26 片 |
| 倾斜度 | 基本垂直 | 顺时针 15° 倾斜 |
| 花心位置 | (212, 204) 偏左 | (256, 232) 居中 |
| 外框色 | `#b8a080` | `#c8b090` |

## 结构细节

- **花朵朝向**：侧视图，面向右侧，花心位于 (256, 232)
- **花瓣层次**：背层 8 片（暗金）+ 中层 10 片（金色）+ 前层 8 片（亮金）
- **倾斜处理**：通过 `<g transform="translate(44, 28) rotate(15, 212, 204)">` 实现

# Select 下拉选择

## 用途
从预定义选项列表中选择一个值。

## 视觉样式

| 属性 | 值 |
|------|-----|
| 高度 | 36px |
| 背景 | `--bg-input` |
| 文字色 | `--text-primary` |
| 边框 | `--border-default`，1px solid |
| 圆角 | `--radius-md`（8px） |

### 交互状态

| 状态 | 样式 |
|------|------|
| 默认 | 灰色边框 + 右侧下拉箭头 |
| 悬停 | 边框变亮 |
| 聚焦 | 边框品牌色 |
| 禁用 | 不透明度 50% |

### 下拉面板

| 属性 | 值 |
|------|-----|
| 背景 | `--bg-elevated` |
| 边框 | `--border-default` |
| 阴影 | `--shadow-lg` |
| 圆角 | `--radius-md` |
| 最大高度 | 240px（可滚动） |

### 选项状态

| 状态 | 样式 |
|------|------|
| 默认 | 行高 36px，内边距 8px 12px |
| 悬停 | `--bg-card-hover` |
| 选中 | 左侧 `--color-primary` 勾选图标 |
| 禁用 | 灰色文字，不可选 |

## 其他变体

- **多选 MultiSelect**: Tag 形式显示已选项，右侧 X 可移除
- **搜索筛选**: 下拉面板内置输入框，实时过滤选项

# Card 卡片

## 用途
内容容器，将相关信息分组展示。

## 视觉样式

| 属性 | 值 |
|------|-----|
| 背景 | `--bg-card` |
| 边框 | `--border-default`，1px solid |
| 圆角 | `--radius-lg`（12px）|
| 内边距 | `--space-lg`（24px）|
| 阴影 | `--shadow-sm` |

### 交互状态

| 状态 | 样式 |
|------|------|
| 默认 | 基础阴影 |
| 悬停（可点击时） | 阴影提升至 `--shadow-md`，边框微亮 |

## 结构

```
┌─────────────────────────────┐
│  [图标] 标题          [操作] │  ← card-header
│─────────────────────────────│
│                              │
│  内容区域                     │  ← card-body
│                              │
│─────────────────────────────│
│  底部信息 / 统计数字          │  ← card-footer（可选）
└─────────────────────────────┘
```

### 各部分

| 部分 | 样式 |
|------|------|
| **header** | 下边框 `--border-default`，内边距 16px 24px |
| **body** | 内边距 24px |
| **footer** | 上边框 `--border-default`，内边距 12px 24px，`--text-secondary` 文字 |

## 尺寸变体

| 变体 | 用途 |
|------|------|
| **Stat Card** | 仪表盘统计卡片，宽高比灵活，数字超大 |
| **Game Card** | 游戏卡片，含封面图区域（120px 高渐变色块） |
| **Plugin Card** | 插件卡片，含图标 + 名称 + 简短描述 |

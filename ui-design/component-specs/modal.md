# Modal 弹窗

## 用途
在保留上下文的同时，聚焦处理特定任务或确认信息。

## 视觉样式

| 属性 | 值 |
|------|-----|
| 背景 | `--bg-elevated` |
| 边框 | `--border-default` |
| 圆角 | `--radius-xl`（16px）|
| 阴影 | `--shadow-xl` |
| 最小宽度 | 420px |
| 最大宽度 | 680px |

### 遮罩

| 属性 | 值 |
|------|-----|
| 背景 | `rgba(0, 0, 0, 0.6)` |
| z-index | `--z-modal-backdrop` |

## 结构

```
┌─────────────────────────────────┐
│  标题                    [✕]    │  ← modal-header
│─────────────────────────────────│
│                                 │
│  内容区域                        │  ← modal-body
│                                 │
│─────────────────────────────────│
│  [取消]                  [确认]  │  ← modal-footer
└─────────────────────────────────┘
```

### 各部分

| 部分 | 样式 |
|------|------|
| **header** | 内边距 16px 24px，h3 标题，文字 `--text-primary` |
| **close** | 右上角 Ghost 风格 X 按钮，32x32 |
| **body** | 内边距 24px，文字 `--text-secondary` |
| **footer** | 内边距 16px 24px，右对齐按钮组，间距 8px |

## 尺寸变体

| 变体 | 宽度 |
|------|------|
| sm | 360px |
| md（默认） | 520px |
| lg | 680px |
| fullscreen | 90vw x 90vh |

## 交互细则

- 点击遮罩关闭（可配置禁用）
- 按 Esc 关闭
- 关闭时内容区域有轻微缩放淡出（150ms）
- 打开时内容不可滚动（`overflow: hidden` 注入 body）

# Table 表格

## 用途
展示结构化数据的行列列表。

## 视觉样式

| 属性 | 值 |
|------|-----|
| 背景 | 透明（继承卡片背景） |
| 圆角 | `--radius-md`（8px）|
| 边框 | `--border-default` |

### 行

| 状态 | 样式 |
|------|------|
| 默认 | 背景透明 |
| 悬停 | `--bg-card-hover` |
| 选中 | 左侧 3px `--color-primary` 竖条 + 浅色背景 |
| 斑马纹（可选） | 偶数行 `rgba(255,255,255,0.02)` |

### 单元格

| 属性 | 值 |
|------|-----|
| 内边距 | 12px 16px |
| 文字 | `--text-body` |
| 垂直对齐 | center |

### 表头

| 属性 | 值 |
|------|-----|
| 背景 | `--bg-card-hover` |
| 文字 | `--text-secondary`，`--weight-medium` |
| 排序指示器 | 可点击列显示 ↑↓ 箭头 |
| 下边框 | 2px `--border-default` |

### 空状态

- 居中显示图标 + "暂无数据" 文字
- 图标 48px，`--text-tertiary`

## 交互细则

- 列宽可拖拽调整（可选）
- 支持多列排序
- 分页控制在表格底部

# Tabs 标签页

## 用途
在有限空间内切换不同内容视图。

## 视觉样式

### 变体一：线型（默认）

```
[ 活动标签 ]  [ 标签二 ]  [ 标签三 ]
─────────────────────────────────────
```

| 状态 | 样式 |
|------|------|
| 默认 | 文字 `--text-secondary`，底部 2px 透明线 |
| 悬停 | 文字 `--text-primary` |
| 激活 | 文字 `--color-primary`，底部 2px `--color-primary` 线 |
| 禁用 | 文字 `--text-tertiary`，不可点击 |

### 变体二：卡片型

```
┌──────────┐ ┌──────────┐ ┌──────────┐
│ 活动标签 │ │  标签二  │ │  标签三  │
└──────────┘ └──────────┘ └──────────┘
```

| 状态 | 样式 |
|------|------|
| 默认 | `--bg-card` 背景，`--border-default` 边框 |
| 激活 | 背景 `--bg-elevated`，上边框 `--color-primary` 2px |

### 通用属性

| 属性 | 值 |
|------|-----|
| 高度 | 40px |
| 内边距 | 8px 16px |
| 字号 | 14px |
| 间距 | 0（紧挨） |
| 圆角 | 顶部 `--radius-md`（卡片型） |

## 交互细则

- 标签数量过多时支持左右滚动（隐藏滚动条）
- 切换时内容区有轻微淡入动画（150ms）

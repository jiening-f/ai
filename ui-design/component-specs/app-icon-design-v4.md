# 游戏全能脚本 — 桌面应用图标设计规格 v4

## 设计概念

**油画风格侧视花朵**，明亮暖色画布，金色层叠花瓣。

告别 v3 的扁平风格，转向油画写实的视觉语言。花朵采用侧视角度、面向右侧的构图，通过 14+ 层渐变花瓣叠加营造油画的笔触层次感。明亮暖色背景让图标在桌面环境中更醒目友好。

## 图标规范

| 属性 | 值 |
|------|-----|
| 尺寸 | 512×512 px（源 SVG 可缩放） |
| 形状 | 圆角方形（r=96px / 512px） |
| 风格 | 油画写实，暖色调，亮色背景 |
| 构图 | 花朵侧视朝向右侧，花茎自然弯曲 |

## 色彩体系

```
画布背景:     #f5ead6 → #d4bea0   明亮暖色画布
背景花瓣:     #7a5a1e → #3d2810   暗金色（远景层）
中间花瓣:     #e8c840 → #8a6815   金色（主体层）
前景花瓣:     #f5d850 → #b07a15   亮金色（焦点层）
花瓣高光:     #fff5b0 → 透明       高光笔触
花萼绿色:     #4a6a28 → #203a12   深绿色
茎叶主色:     #5a7a36 → #1e3612   橄榄绿
叶子高光:     #8aac3e              叶脉光泽
图标外框:     #b8a080 → #d4c0a0   暖金色描边
```

## 图层结构

```
┌──────────────────────────┐
│    明亮暖色画布     │
│  ┌────────────────────┐  │
│  │  背景层花瓣（5片）    │  │  ← 暗金色，花头后方
│  │  中间层花瓣（6片）    │  │  ← 金色，花主体
│  │  前景层花瓣（5片）    │  │  ← 亮金色，视觉焦点
│  │     花萼（绿色）      │  │  ← 花头基部
│  └────────────────────┘  │
│         │                │
│       花茎（弯曲）        │
│      ↙     ↘             │
│    叶子    叶子           │
│  （左）    （右）         │
└──────────────────────────┘
```

## 结构细节

- **花朵朝向**：侧视图，面向右侧，花心位于 (210, 200) 附近
- **花瓣层次**：背层 5 片（暗金）+ 中层 6 片（金色）+ 前层 5 片（亮金）
- **花心**：花瓣交汇处自然暗部阴影
- **花茎**：从花萼延伸至底部，宽约 8px，自然左弯
- **叶子**：左叶 (209, 335) 向左展开 + 右叶 (203, 385) 向右展开，带叶脉线
- **外框**：双描边（暖金 2.5px + 浅金 0.8px），增强图标边界可见性

## 对比度说明

明亮暖色背景在浅色和深色桌面环境中均能自然融合。金色花瓣在暖底上保持清晰的对比度。外框双描边在任何桌面背景下都不会"消失"。

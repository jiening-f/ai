# 游戏全能脚本 — 桌面应用图标设计规格 v4

## 设计概念

**油画风格侧视花朵**，暗色暖底画布，金色层叠花瓣。

告别 v3 的扁平风格，转向油画写实的视觉语言。花朵采用侧视角度、面向右侧的构图，通过 14+ 层渐变花瓣叠加营造油画的笔触层次感。暗色暖底传达沉稳、强大的产品气质。

## 图标规范

| 属性 | 值 |
|------|-----|
| 尺寸 | 512×512 px（源 SVG 可缩放） |
| 形状 | 圆角方形（r=96px / 512px） |
| 风格 | 油画写实，暖色调，暗色背景 |
| 构图 | 花朵侧视朝向右侧，花茎自然弯曲 |

## 图层结构

```
┌──────────────────────────┐
│    暗色暖底 + 晕影效果     │
│  ┌────────────────────┐  │
│  │  背景层花瓣（5片）    │  │  ← 暗金色，花头后方
│  │  中间层花瓣（6片）    │  │  ← 金色，花主体
│  │  前景层花瓣（5片）    │  │  ← 亮金色，视觉焦点
│  │     花蕊 + 发光      │  │  ← 暖橙色，花粉点
│  │     花萼（绿色）      │  │  ← 花头基部
│  └────────────────────┘  │
│         │                │
│       花茎（弯曲）        │
│      ↙     ↘             │
│    叶子    叶子           │
│  （左）    （右）         │
└──────────────────────────┘
```

## 结构细节

- **花朵朝向**：侧视图，面向右侧，花心位于 (210, 200) 附近
- **花瓣层次**：背层 5 片（暗金）+ 中层 6 片（金色）+ 前层 5 片（亮金）
- **花蕊**：椭圆主体 (18×16) + 柔光效果 + 8 个花粉点
- **花茎**：从花萼延伸至底部，宽约 8px，自然左弯
- **叶子**：左叶 (209, 335) 向左展开 + 右叶 (203, 385) 向右展开，带叶脉线
- **外框**：双描边（深色 2.5px + 金色 0.8px），增强图标边界可见性

## 对比度说明

暗色暖底在深色桌面环境（Windows 暗色模式、macOS 暗色主题）中融合自然，金色花瓣在暗底上具有高对比度。外框双描边确保在浅色桌面上不会"消失"。
